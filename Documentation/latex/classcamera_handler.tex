This method allows to change the angle of the position of the camera by inputting the said angle in degrees. 

The angle can vary from 0 degrees to the value of {\bfseries{angle\+View\+Amplitude}} , 0 when the camera is facing the right side of the robot and the value of {\bfseries{angle\+View\+Amplitude}} when it is facing the left side.
