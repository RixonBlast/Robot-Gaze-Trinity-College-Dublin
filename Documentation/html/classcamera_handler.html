◆ setCameraAngle()

void cameraHandler.setCameraAngle ( float  angle)

This method allows to change the angle of the position of the camera by inputting the said angle in degrees.

The angle can vary from 0 degrees to the value of angleViewAmplitude , 0 when the camera is facing the right side of the robot and the value of angleViewAmplitude when it is facing the left side.

Parameters
angleThe angle, in degrees, desired for the position of the camera.
Returns
It returns nothing </return>

◆ setCameraMovementEnabled()

void cameraHandler.setCameraMovementEnabled ( bool  value)

This methods allows to enable or disable the camera movement. When the camera movements are disabled the camera won't be able to move using any of the two functions setCameraAngle or setCameraState, otherwise it will be possible.

Parameters
valueIf value is set to true the camera's movements will be enabled, if not they will be disabled.

◆ setCameraState()

void cameraHandler.setCameraState ( int  cameraState)

This method allows to change the angle of the position of the camera only by inputting an integer number between 0 and the value of nbCameraState - 1 .

For example if the value of nbCameraState is 5 and the value of angleViewAmplitude is 180 then the camera will be positioned the following way depending on the input of the method:

cameraState 's value angle of position of the camera in degrees
0 0
1 45
2 90
3 135
4 180

Note that when the angle of position of the camera is 0 degrees then it will be facing the right side of the robot, when its value is 180 degrees it will be facing the left side of the robot.

Parameters
cameraStateThis value has to be included between 0 and nbCameraState - 1 and will determine the camera's position around the robot.
Returns
It returns nothing
