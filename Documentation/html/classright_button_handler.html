Detailed Description

Handles the right button of the main canvas used to give instruction to the user. This class only handles the visual aspect of the button. The right button is used to go to the next slide. It appears white when the mouse is hovering it and in a dark blue color when it is not hovered by the mouse. The right button disappears when there is no page to go to.
